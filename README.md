# C++ Modules

These modules purpose is to teach basic C++ skills.

## Skills and knowledge learnt

- classes
- function overloading
- orthodox canonical form
- inheritance
- polymorphism
- try/​catch & exceptions
- casting
- templates
- containers
- iterators

## Instructions

To compile each excercise, run
```
make
```

To execute each program, run
```
./*program_name* *arguments_if_needed*
```
